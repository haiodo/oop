# Основные различия синтаксиса C++ и Java

## Слайд 1: Заглавный слайд

**Название:** Основные различия синтаксиса C++ и Java

**Подзаголовок:** Сравнение двух популярных языков программирования

---

## Слайд 2: Введение

- Краткое введение в тему.
- Цель презентации: сравнить синтаксические особенности C++ и Java.

---

## Слайд 3: Управление памятью

### C++

- Ручное управление памятью с помощью `new` и `delete`.

```cpp
int* ptr = new int;
delete ptr;
```

### Java

- Автоматическое управление памятью через сборку мусора (Garbage Collector).

```java
int value = 10; // Автоматическая сборка мусора
```

---

## Слайд 4: Типы данных и переменные

### C++

- Поддержка указателей, структур, перечислений.

```cpp
int a = 10;
double b = 20.5;
```

### Java

- Все объекты наследуются от `Object`, нет явных указателей.

```java
int a = 10;
double b = 20.5;
```

---

## Слайд 5: Наследование и полиморфизм

### C++

- Поддержка множественного наследования.

```cpp
class Base {};
class Derived : public Base {};
```

### Java

- Поддержка только однородного наследования, интерфейсы для полиморфизма.

```java
interface Animal {}
class Dog implements Animal {}
```

---

## Слайд 6: Исключения

### C++

- Использует `try`, `catch` и `throw`.

```cpp
try {
    // код, который может вызвать исключение
} catch (const std::exception& e) {
    // обработка исключения
}
```

--- 
### Java

- Также использует `try`, `catch` и `throw`, но имеет более строгую систему проверки исключений.

```java
try {
    // код, который может вызвать исключение
} catch (Exception e) {
    // обработка исключения
}
```

---

## Слайд 7: Массивы и контейнеры

### C++

- Использует массивы и STL (Standard Template Library).

```cpp
std::vector<int> vec = {1, 2, 3};
```

### Java

- Использует массивы и коллекции из `java.util`.

```java
List<Integer> list = Arrays.asList(1, 2, 3);
```

---

## Слайд 8: Функции и методы

### C++

- Поддержка перегрузки функций и операторов.

```cpp
void print(int x) {}
void print(double x) {}
```

### Java

- Методы являются частью классов, нет перегрузки операторов.

```java
class Example {
    void print(int x) {}
    void print(double x) {}
}
```

---

## Слайд 9: Модули и пакеты

### C++

- Использует заголовочные файлы (`.h`) и библиотеки.

```cpp
#include "example.h"
```

### Java

- Использует пакеты (`package`) и модули (JDK 9+).

```java
package com.example;
import java.util.List;
```
