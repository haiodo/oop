# Лямбда-выражения в C++ и Java

## Слайд 1: Заглавный слайд

**Название:** Различия между лямбда-выражениями в C++ и Java

**Подзаголовок:** Сравнение функциональных возможностей двух популярных языков программирования

## Слайд 2: Введение

- Лямбда-выражения позволяют создавать анонимные функции.
- Цель презентации: сравнить синтаксис и возможности лямбда-выражений в C++ и Java.

---

## Слайд 3: Лямбда-выражения в C++

### Синтаксис

```cpp
[capture list](parameters) -> return_type {
    function body;
}
```

### Пример

```cpp
auto add = [](int a, int b) { return a + b; };
std::cout << add(3, 4); // Выводит 7
```

---

## Слайд 4: Лямбда-выражения в Java

### Синтаксис

```java
(parameters) -> {
    function body;
}
```

### Пример

```java
BiFunction<Integer, Integer, Integer> add = (a, b) -> a + b; 
System.out.println(add.apply(3, 4)); // Выводит 7
```

---

## Слайд 5: Типы возвращаемого значения

### C++

- Явное указание типа возврата обязательно для лямбда-выражений с не-`void` типом возврата.

```cpp
auto add = [](int a, int b) -> int { return a + b; };
```

### Java

- Тип возврата выводится из контекста.

---

## Слайд 6: Захват переменных

### C++

- Возможность захвата переменных по значению или по ссылке.

```cpp
int x = 10;
auto lambda = [x]() { std::cout << x; }; // Захват по значению
auto lambdaRef = [&x]() { x = 20; };    // Захват по ссылке
```

### Java

- Лямбда-выражения в Java захватывают переменные по значению.

```java
int x = 10;
Runnable lambda = () -> System.out.println(x); // Захват по значению
```

---

## Слайд 7: Применение лямбда-выражений

### C++

- Используются в STL (Standard Template Library).

```cpp
std::vector<int> vec = {1, 2, 3};
std::for_each(vec.begin(), vec.end(), [](int n) {
    std::cout << n << " ";
});
```

### Java

- Используются с интерфейсами функциональных программирования, такими как `Function`, `Predicate`, `Consumer`.

```java
List<Integer> list = Arrays.asList(1, 2, 3);
list.forEach(n -> System.out.print(n + " "));
```

## Слайд 8: Типы лямбда-выражений в C++

### Пример использования std::function

```cpp
std::function<int(int, int)> add = [](int a, int b) { return a + b; };
std::cout << add(3, 4); // Выводит 7
```

---

## Слайд 9: Типы лямбда-выражений в Java

### Пример использования интерфейсов функционального программирования

```java
Function<Integer, Integer> square = x -> x * x;
System.out.println(square.apply(5)); // Выводит 25
```
